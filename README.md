## 🚀 Click To Clean (CTC) – Data Cleaning App

This project is an interactive **data cleaning web application** built using **Streamlit**.  
**CTC (Click To Clean)** is designed to simplify and speed up the data cleaning process through an intuitive, user-friendly interface.

👉 **Live App:** https://ctcdata-ns38.streamlit.app/

---

## 📊 Overview

Data cleaning is one of the most time-consuming steps in any data workflow.  
This application helps users quickly clean and prepare datasets without writing complex code.

With just a few clicks, users can:

- Handle missing values  
- Remove duplicates  
- Standardize and transform data  
- Prepare datasets for analysis or machine learning  

---

## ✨ Features

### 📁 Data Input
- Upload datasets (CSV format)  
- Preview raw data instantly  

### 🧹 Data Cleaning Tools
- Handle missing/null values  
- Remove duplicate records  
- Rename and modify columns  
- Data type adjustments  

### 🔧 Data Transformation
- Basic data formatting and standardization  
- Column-level operations  
- Prepare clean datasets for downstream use  

### ⚡ Interactive Experience
- Click-based cleaning actions (no coding required)  
- Real-time updates to dataset  
- Simple and clean UI powered by Streamlit
